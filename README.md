# Roy Huang - Crypto & AI Portfolio

A modern, interactive portfolio website showcasing Roy Huang's expertise in cryptocurrency analysis and AI content production.

## 🚀 Features

- **Real-time BTC Price Display**: Live cryptocurrency market data integration
- **Interactive Bento Grid Layout**: Modern card-based design with hover effects
- **Article Management System**: Dynamic content loading from Notion API
- **Photo Gallery Modal**: Beijing journey showcase with smooth animations
- **Responsive Design**: Optimized for desktop and mobile devices
- **Dark Theme**: Cyberpunk-inspired aesthetic with purple accents

## 🛠 Tech Stack

- **React 19** - Latest React with modern hooks
- **Framer Motion** - Smooth animations and transitions
- **Tailwind CSS** - Utility-first CSS framework
- **Lucide React** - Beautiful icon library
- **Vite** - Fast build tool and development server

## 🔧 Development

### Prerequisites
- Node.js 18+ 
- npm or yarn

### Installation
```bash
# Clone the repository
git clone <repository-url>

# Navigate to project directory
cd my-portfolio

# Install dependencies
npm install --legacy-peer-deps

# Start development server
npm run dev
```

### Build for Production
```bash
npm run build
```

### Preview Production Build
```bash
npm run preview
```

## 🌐 API Integration

The website integrates with several external APIs:

- **Binance API**: Real-time BTC/USDT price data
- **Notion API**: Dynamic article content loading
- **Custom Vercel API**: Article detail fetching

## 📱 Responsive Breakpoints

- **Mobile**: < 768px
- **Tablet**: 768px - 1024px  
- **Desktop**: > 1024px

## 🎯 Key Sections

1. **Hero Section**: Personal introduction with live BTC price
2. **Research Analysis**: Dynamic article loading from Notion
3. **Tools Section**: Web3 and AI tool links
4. **Photo Gallery**: Beijing journey showcase
5. **Social Links**: Contact and social media connections

## 🔒 Security Features

- **CORS Handling**: Proper cross-origin request management
- **Error Boundaries**: Graceful error handling
- **Fallback Content**: Local cache when APIs are unavailable
